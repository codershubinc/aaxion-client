# Aaxion v0.0.1 - Initial Release

> ⚠️ **EARLY RELEASE WARNING:** This is an initial early alpha release. The software is actively under development and you may encounter bugs, incomplete features, or instability.

Welcome to the first official release of the Aaxion Client (v0.0.1). This release establishes the core foundation for a seamless, cross-platform media and file streaming experience, wrapping specialized web functionality inside a lightweight Tauri desktop app.

## 🌟 Key Features

### 1. Unified Cross-Platform Architecture

- **Tauri Desktop Client:** A high-performance desktop entry point for Windows/Linux/macOS with a completely borderless, transparent, custom drag-and-drop window design.
- **Web Fallback:** Clean Next.js architecture allowing the app to run perfectly via browser when the desktop environment isn't preferred.

### 2. Aaxion Drive (Local Cloud Explorer)

- **File Exploration System:** Browse, navigate, and manage remote storage smoothly with a responsive file grid and list view.
- **Media Preview:** First-class integrated image previews (`ImagePreview.tsx`) without needing to launch external software.
- **File Uploads:** Managed upload workflow via standard `UploadModal` to easily move local desktop files to Aaxion servers.
- **Quick Access:** Bookmark-style quick access for rapid navigation between commonly used directories.

### 3. Aaxion Streamer (Media Center)

- **Movies Library:** Beautiful grid layouts for exploring remote film files. Auto-fetches and caches metadata via the OMDB API (`useOmdbCache`).
- **Series / TV Shows Library:** Hierarchical series layout, supporting adding series, diving into particular shows, and parsing episodic content.
- **Administrative Adding Tools:** Intuitive manual "Add Movie" and "Add Series/Episode" forms for manually curating the library.

### 4. VLC Media Integration

- **Native Casting to VLC:** Unique one-click native streaming. Bypasses browser transcoders by casting remote video links seamlessly to the local VLC Media Player.
- **VLC Remote Control:** Built-in web overlay and remote handlers to play, pause, seek, and adjust volume on the local standard VLC instance directly through the Aaxion interface.

### 5. Seamless Server Networking

- **Auto IP Discovery:** The companion `useDiscovery` and local IP fetchers allow automated connecting to the local Aaxion Server backend without manual IP typing.
- **Auth System:** Role-based logic requiring secure connection keys using the integrated robust Login interfaces and Auth Overlays.

### 6. Desktop Specifics

- **Borderless Floating TitleBar:** Fully custom Tauri-driven top navigation bar allowing minimization, maximization, and closing, completely overriding default OS decorators.
- **Linux Native Support:** Support for AppImage distribution with install scripts and desktop shortcut generators.
- **Optimized Performance:** Leverages Next.js + standalone fast Rust-based Tauri backend for extremely low memory consumption.
